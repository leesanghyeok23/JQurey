		- 자바스크립트 문법으로 만들어진 다양한 함수들을 작성 해 놓은 자바스크립트 파일.

		방법1.   JQuery라이브러리 파일 .js를 다운로드해서 사용하는 방법 
				
				순서1. http://jquery.com 접속 해서  .js를 다운 받는다.  또는 
					  
					  
					  jquery.js파일
					  -> 소스에 들여쓰기, 줄 바꿈, 주석까지 정리된 비압축파일로
					         제이쿼리 문법을 이용하여 개발할때 유용함
					  
					  jquery.min.js파일
					  -> 소스에 들여쓰기, 줄 바꿈, 주석이 무시된 압축파일로
					  	 제이쿼리 문법을 이용하여 개발하는 용도로 적합하지 않습니다.
					  	 하지만 파일 용량이 작아 웹브라우저의 로딩 속도가 빠릅니다. 
					  
					  js폴더 생성후 두개의 파일중 하나를 복사 둡니다.

					  <script src="js/jquery-3.7.0.js"></scrip> 를

			2. 문서 객체 조작메소드 ?
			   - 문서 객체를 생성, 복제, 삭제, 속성 변환하는 메소드.
			   - 
			   	  예1.선택한 요소에 새요소를 만듭니다. - 새 요소 생성
					 	<div></div>         ->     <div><p>내용</p></div>	   
				  
				  	예2.요소를 복제하여 선택한 요소에 삽입합니다.  - 복제 
				    <div>내용1</div>     ->     <div>내용1</div>  
				     														 <div>내용1</div>
			   
						예3.선택한 요소를 삭제합니다. - 삭제
						<div><p>내용</p></div>   -> <div></div>
			         
						예4.선택한 요소의 속성 값을 변경 또는 추가 합니다.그리고 기존의 속성과 값을 삭제도 합니다. - 속성 변환
						
							속성값을 변경 한 경우 
							<img src="btn_on.gif" alt="버튼1" />    -> <img src="btn_off.gif" alt="버튼2" />    
										
							속성값을 추가 한 경우     
							<img src="btn_on.gif" />    -> 	<img src="btn_on.gif" alt="버튼1" />
						
							속성값을 삭제한 경우     
							<img src="btn_on.gif" alt="버튼1" />    -> 	<img src="btn_on.gif" />

03이벤트 폴더

        참고. 
        이벤트?
          - 사이트 방문자가 취하는 모든 동작.
          
         이벤트 핸들러?
          - 특정요소에서 발생하는 이벤트를 처리하기 위한 함수.
                 
         이벤트 객체 ?
          - 이벤트 핸들러 함수 function(event)는 이벤트객체를 event매개변수로 전달 받을 수 있습니다.
						이렇게 전달 받은 이벤트 객체를 이용하여 이벤트가 발생한 요소를 다시 선택해서 가져와 이벤트를 처리  할수 있습니다. 
						이때 매개변수 event로 전달 받은 이벤트 객체란?  
						이벤트가 발생한 요소의 정보가 저장된 객체를 이벤트객체 라고 합니다.
                      
		3. 이벤트 등록 메소드?
		 -  사이트 방문자가 선택한 요소에서 어떤한 동작(이벤트)를  하기 전에 미리 선택한 요소에 동작(이벤트)를 등록 하는 메소드.
		 -  
		    종류
		 	 
		 	  이벤트 등록 메서드 종류 
				  이벤트 등록 메소드의 종류.jpg 파일 표 참고.
		 
		 		3-1. 단독 이벤트 등록메소드

			 효과 및 애니메이션 메서드
			
			[효과 메서드] : 선택한 요소를 역동적으로 숨겼다가 보이게 만드는 기능을 가진 메서드
			
			[효과 메서드 종류]
			hide() : 노출되어 있는 요소를 숨김니다.
			show() : 숨겨져 있던 요소를 노출 시킵니다.
			
			------------------ jquery_effect_1.html 파일 실습
			
			toggle() : 숨겨져 있던 요소는 노출시키고, 노출되어 있던 요소는 숨깁니다.
			
			------------------ jquery_effect_2.html 파일 실습
			
			fadeln() : 숨겨져 있던 요소가 점점 선명해지면서 노출 됩니다.
			fadeOut() : 노출되어 있던 요소가 점점 투명해 지면서 숨겨 집니다.
			fadeToggle() : 노출되어 있던 요소는 점점 투명해 지며 사라지고 , 사라져 있던 요소는 선명해지면서 노출됩니다.
			
			
			------------------ jquery_effect_3.html 파일 실습
			
			
			
			fadeTo() : 노출되어 있던 요소에 투명도를 지정한 만큼 숨깁니다.
			
			
			------------------ jquery_effect_4.html 파일 실습
			
			
			slideDown() : 숨겨져 있던 요소가 밑으로 펼쳐지며 노출됩니다.
			slideUp() : 노출되어 있던 요소를 위로 접으며 숨깁니다.
			slideToggle() : 숨겨져 있던 요소는 아래로 펼쳐지며 노출되고, 노출되어 있던 요소는 위로 접으면서 숨깁니다.
			
			
			------------------ jquery_effect_5.html 파일 실습
			
			
			
			애니메이션 동작 설정 하는 메소드.

  			제이쿼리 플러그인이란?
  				여러 개발자가 드래그 &드롭, 슬라이드배너, 툴팁 기능 등을 구현해 놓은 프로그램을 자바스크립트 파일로 제공하는
  				제이쿼리 라이브러리.
  			
  			제이쿼리 플러그인 사용법
  				사용하고자 하는 기능의 플러그인을 검색하여 내려 받고 웹 문서에 연동하여 선택한 요소에 기능만 적용하면 
  				간단한 코드 삽입만으로도 복잡하고 어려운 기능을 쉽게 구현할수 있습니다.
  				
  			제이쿼리 플러그인 적용 문법

  						1. 제이쿼리 라이브러리 연동
